# 정보들

내가 앞으로 프로그래밍에서 유용하게 이용할 윈도우 단축키 정보나 간단한 프로그래밍 용어들을 여기에 정리하도록 한다.

## 프로그래밍 용어

| 용어 | 의미    |
| ---- | ------- |
| `\n` | `Enter` |
|      |         |



## 윈도우 단축키

- 유용한 윈도우 단축키들 중 내가 몰랐던 것들을 저장

| 명령어              | 설명                                   |
| ------------------- | -------------------------------------- |
| ` win + shift + s ` | 스크린샷. 찍고 바로 paste 가능한 장점. |
|                     |                                        |
